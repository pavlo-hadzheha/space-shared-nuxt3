Установіть Node.js v16.13.2 або вище за посиланням https://nodejs.org/uk/

Відкрийте дві сесії термінала та, знаходячись в кореневій папці проекту, введіть
наступні команди:

1) Термінал #1 

cd BACKEND
npm install
npm run start

2) Термінал #2

cd FRONTEND
npm install
npm run dev

Після цього, у бразері, заходимо за посиланням http://localhost:3000
